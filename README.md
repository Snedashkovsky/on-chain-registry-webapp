# On-Chain Registry webApp
## build
```bash
npm run build
```

## deploy
by local [ipfs node](https://docs.ipfs.tech/install/)
```bash
ipfs daemon
npm run deploy
```
or by [ipfs-deploy](https://github.com/ipfs-shipyard/ipfs-deploy)
```bash
npm install -g ipfs-deploy
npm run build
ipfs-deploy build
```
